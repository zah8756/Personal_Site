This project is curenlty broken but the tech behind it is solid. It was created in 2020

This application allowed users to connect to a platform where they could bet virtual currency in rock-paper-scissors games. I implemented a lobby system using Socket.IO, enabling users to create lobbies in real-time and see who was looking for a match. User data was encrypted and then stored using MongoDB.
